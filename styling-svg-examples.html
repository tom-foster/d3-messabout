Layering and Drawing Order

SVG's do not support layering so the z-index can't be used.

Transparency

It's useful to use transparency if we're going to overlap using svgs.

You can use either rgba or opacity they behave slightly differently.

Using rgba()

Using rgba() on stroke has to be applied independently

As you can see the stroke looks like two seperate 5px strokes.

To apply transpency to an entire element set opacity attribute.

Opacity can be applied to rgba as well
