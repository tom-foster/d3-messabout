For version 4, of d3.js scale is defined as scaleLinear.

If your using chrome dev tools you'll see it pop up as you type in console.

var scale = d3.scaleLinear()
.domain([100, 500])
.range([10, 350]);
//then try out these
scale(100); //returns 10
scale(300); //returns 180
scale(500); //returns 350
